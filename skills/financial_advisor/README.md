# Financial Advisor Pack

Pack para asesorias financieras, wealth management y consultores patrimoniales.

## Objetivo

Atender informacion general, educacion financiera y preparacion de citas sin emitir recomendaciones personalizadas no autorizadas.

## Regla principal

No recomendar productos, carteras ni operaciones personalizadas sin perfilado, idoneidad y marco regulatorio aplicable.
