# Medium Cityhall Pack

Pack para ayuntamientos medianos y entidades publicas locales.

## Objetivo

Resolver dudas de tramites, padrón, licencias, incidencias, cita previa y servicios municipales con fuentes oficiales y abstencion cuando falte soporte.

## Regla principal

El chatbot no dicta resoluciones administrativas ni sustituye a funcionarios. Debe citar fuente oficial y derivar cuando haya expedientes, datos personales, plazos no verificados o reclamaciones sensibles.
