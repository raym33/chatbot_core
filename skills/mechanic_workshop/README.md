# Mechanic Workshop Pack

Pack para talleres mecanicos, concesionarios y servicios postventa.

## Objetivo

Atender citas, diagnostico preliminar no definitivo, presupuestos, garantias, recambios y seguimiento de reparaciones.

## Regla principal

No prometer diagnosticos, precios, tiempos ni coberturas de garantia sin inspeccion o sistemas autorizados.
