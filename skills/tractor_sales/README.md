# Tractor Sales Pack

Pack para concesionarios de tractores, maquinaria agricola y postventa.

## Objetivo

Ayudar con catalogo, configuracion, financiacion orientativa, garantias, recambios, taller y postventa.

## Regla principal

No inventar stock, precios, descuentos, financiacion ni garantias. Todo dato comercial variable debe venir de CRM/ERP/catalogo autorizado.
