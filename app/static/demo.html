Chatbot Core

Grounded multi-domain assistant prototype

Technical demo for a reusable assistant stack with hybrid RAG, policy-based abstention, sector packs, tool calling, and open-source voice capabilities.

RAG Hybrid lexical + semantic retrieval
Safety Verification and abstention-first behavior
Voice Whisper STT and edge-tts TTS

Suggested checks

  • Ask a factual question that should be answered from the corpus.
  • Ask for unsupported or missing data and verify abstention.
  • Trigger a sensitive query and verify safe escalation behavior.
  • Use the voice loop to validate STT, answer generation, and TTS.
  • Inspect /v1/cluster when testing distributed workers.
